    > LCS

    Reference Video: SWOS

    • Work Done:
      • Implementation of Lesson Content
      • Solving Bugs with Multiplayer Features
      • Optimization of Unreal Physics Cables
        • Tension Settings
        • Reduction of Rebuild Calls
        • Number of Collision Samples
      • Rendering Optimization
        • Auto-Calculated Bounding-Box Distance Culling Tool
          • This tool was built to quickly set distance culling on all objects within a given scene based on bounding box, and average view distance.
        • Batch Editing Actor Render Settings
          • Editor Utility Widget
            • Search a level for actors with specific render properties
            • Adjust render properties of all selected actors in a level
            • Modify asset ref. of those actors to propagate to other levels
      • Work Acceleration
        • Socket-Based Asset Placement Tool
          • Problem: Designers were offloading tasks to the Art team who was already a bottleneck because they already had all the reference pictures for placing things in the level.
          • Goals:
            • Reduce time Artists spend in Unreal
            • Automate asset replacement
            • Decouple Art & Design Pipelines
          • Art Workflow Change:
            • Add Empties to Mesh with specific naming convention
              • These will import into Unreal as named sockets for the Static Mesh
              • Placing empties on the mesh is faster than:
                • Opening Unreal
                • Answering Designer Questions
                • Designers Searching Reference Imagery
          • Tool Solution:
            • Editor Utility Widget that builds a list of existing sockets & a list of all interactive actors from Content Browser
            • Feature to automatically place assets with correct tagging & naming conventions
            • Asset sockets support getting the correct in-level position, rotation, and scale for the associated mesh
