    > Helldivers 2

    When I joined this project I was initially tasked with “Improving the Armor Pipeline”. I started by digging through all the documentation they had written regarding the pipeline itself. I did a couple shadow sessions and interviewed artists about what the pain points and slow parts of the process were. In many cases comments led to 2 places, issues with the Stingray Engine, and needing to modify many, many text files.

    • Responsibilities
      • Improve the Armor Implementation Pipeline
        • Documentation & Clarifying Process
        • Engine Modification Improving Workflows
        • Pitches for Pipeline Improvements
        • Extending Python Tools
        • Troubleshooting Implementation for Artists
      • Improve Bulletpainting System
        • Optimized to support more units, larger units, and maintain texel density
        • Documented process for Technical Artists & Artists to better maintain in the future
        • Troubleshooting for Artists
      • Optimize Uber Customization Shaders
      • Develop Feature for Minimap
    • Additional Contributions
      • Feedback & Improvement of Onboarding Process & Documentation
      • Mentor New / Junior Team Members

    > Payday 3

    The team was initially hired to implement VOIP for the game, I joined months after work had started & helped get the first version(s) of VOIP working on PC. I also supported efforts to integrate PlayStation and XBOX. The contract continued into an implementation of the Server Browser, where I had ownership of implementing the design. After estimating the time it would take, we needed to complete the task sooner & I took on delegating work between team members based on dependency & design restrictions.

    • Responsibilities
      • Implement UI Elements for VOIP
      • Implement Server Browser UI
      • Ensure UI meets Sony TRC Compliance
    • Additional Contributions
      • Support Integration with Epic Online Services
      • Troubleshoot Server Authentication & Tokens from Steam & PSN
      • VOIP
        • Troubleshoot VOIP Logic
        • Add Features from Backend to Frontend
      • Ownership of Server Browser UI
        • Delegation to Team Members

    > NDA Project

    We were working on a game that was ported to Mobile & tasked with taking the Mobile port & porting that to Console. We were not allowed to modify the existing UI and the client did not want to maintain 2 separate UI versions. So in order to facilitate the UI we worked on developing a “Console Wrapper” that made the UI feel intuitive with a Controller.

    • Responsibilities
      • Map UI & Player Interactions start to end
        • Build out the scope of work
        • Ensure we were not overlooking implementation details
        • Identify Stability Issues in UI
      • Implementation of UI Features
      • Partial Design of UI/UX Features
    • Additional Contributions
      • Developed Demos & Prototypes for how we could tackle the problem without being intrusive to the client code or UI
      • Aided in the System Design of Input Handling System
      • Aided in Educating Client’s team on Unreal UI best practices for Implementation & Rendering
      • Developed a Localization Solution for the Team to Improve Work Speed

    > Immortals of Aveum

    A First-Person Magic Shooter I joined this project as a Technical Artist with the aim of optimizing their Animation Blueprints. Adjusting all the animations to follow Unreal’s “fast path” system that supports multi-threading of animations. I was also tasked with fixing character Control Rigs & the integration of Control Rig into Animation Blueprints.

    • Responsibilities
      • Implement Fast Path for Animation Blueprints
      • Fix and Optimize Control Rigs
      • Integrate VFX into Cinematics
      • Optimize Physics & Collision
      • Implement Progression Based Skeletal Mesh Swap
    • Additional Contributions
      • Identified specific bug locations in code & offer solutions
        • Because of my Tech Art title I wasn’t allowed to directly contribute to code
      • Took additional Post-Process Animation work off an overloaded Technical Artist’s plate

    > Alien: Rogue Incursion

    This is a VR Alien game and my team was tasked with helping develop the puzzle mini-games initially. After nailing these down we started diversifying the systems we supported. Synth animations and dismemberment as well as weapon combat features. I worked on the implementation for the Smart Gun, auto-targeting, and turret functionality while supporting the development of the Flamethrower and Synth dismemberment.

    • Responsibilities
      • Design and Implement Logic Puzzles
      • Support Development of other Puzzle Mini-Games
      • Implement the Smart Gun Features
    • Additional Contributions
      • Supported Implementation of the Flamethrower
      • Developed a Tool for Modifying “Motherboard Components” like Prefabs
      • Developed a Tool for adding “greebles” to the Motherboard and filling space

    > Sensorium Galaxy

    I joined TLM in the middle of this contract for the initial creation of Sensorium Galaxy, developing a metaverse concept for Virtual Shows. I inherited and built on a system for mapping video as a form of LUT to individual stage pieces. I built a tool that enabled artists to visualize how the video was being divided, what each mesh was assigned, and enabled the easy assignment of new video sections to meshes in the scene.

    • Responsibilities
      • Design & Implement a solution for streaming small 256x256 video(s) as stage VFX
      • Develop a content tool for artists to easily map the video(s) to scene meshes
      • Request specific UV layouts for meshes to support specific VFX
